Création du produit

Pour créer un articles allez sur Catalogue > Produits > Créer (en haut à droite)

Première étape : créer le produit

Dans la première étape, remplissez d'abord les informations importantes (voir capture ci-dessous) puis enregistrez le produit. Cette action permet d'activer certaines fonctionnalités propres au produit.

Par convention, le numéro de référence est du format suivant : xxx-yyy, xxx étant le numéro de la revue et yyy étant le numéro de la page à laquelle l'article commence (p.ex 045-009). Ces données vont servir à la table des matières. Le numéro complet d'une revue (p.ex la revue n° 48) aura une référence en xxx pour 048.

Quand vous enregistrez, vous avez le choix soit de rester sur le produit actuel, soit de retourner vers le listing après la sauvegarde. Pour restez choisissez la disquette plutôt que le vu.

Deuxième étape : définir un prix

Sur l'onglet prix saisissez soit le prix hors taxes, soit le prix TTC. L'autre champ sera automatiquement calculé. Vérifiez en bas que le prix final est bien le bon.

Troisième étape : ajouter des pièces jointes

Transformer en produit avec un fichier privé à télécharger

Si le produit que vous créez est un article destiné à la vente avec un pdf accessible après l'achat, choisissez Produit dématérialisé dans l'onglet Information.

Dans le menu de gauche allez dans Produit dématérialisé. Faites parcourir et choisissez le pdf concerné et donnez lui un nom cohérent. Ce nom sera celui que le fichier aura au moment du téléchargement indépendamment du nom qu'il a lorsque vous le prenez sur votre disque dur.

Ne saisissez pas de date limite d'expiration ni de nombre de téléchargement maximum.

Ajouter un fichier à télécharger gratuitement

Pour ajouter un document, donnez lui un nom dans la partie du haut et parcourez votre disque dur pour le saisir. Enregistrez. Ce dernier apparaît ensuite dans la colonne de droite un peu plus bas.

Un document pouvant être lié à plusieurs produits, celui que vous venez d'ajouter apparaîtra dans la colonne de gauche pour tous les autres produits. Il suffira d'aller l'ajouter à la colonne de droite pour le lier.

Quatrième étape : choisir où afficher le produit

Sur l'onglet associations vous allez pouvoir choisir à quelle(s) thématique(s) associer le produit. Dépliez la navigation en cliquant sur le "+" et cochez autant de catégories que vous le souhaitez. Les produits apparaîtrons quand on cliquera dans la navigation du site.

Pensez à les mettre à la fois dans Thèmes et dans Boutique > Acheter par articles pour des articles unitaires ou dans Boutique > Commande par numéros s'il s'agit de la revue complète.

La catégorie par défaut, n'est pour l'instant pas utilisé. C'est elle qui est utilisée si on souhaite faire des règles de prix pour des promotions. Je préconise de choisir la rubrique adéquate qui se trouve dans Boutique (soit Commande par numéros soit Acheter par articles.

La section articles liés permet de lier la revue à tous les articles qu'elle contient. Cette liaison est unidirectionnelle, ce qui signifie que les articles n'auront pas conscience que la revue est liée à eux. Il faut faire la liaison dans l'autre sens. Les articles unitaires pourront être liés à la revue complète et à d'autres articles similaires.

Vous pouvez taper le nom de l'article désiré dans le champ de recherche mais aussi la référence. Ainsi il est relativement aisé de retrouver tous le articles du numéro 48 en écrivant 048 dans le champ.

Une corbeille permet de délier le produit.

Cinquième étape : choisir une image de couverture

Sur l'onglet images choisissez à votre guise une image pour représenter le produit. Vous pouvez en choisir plusieurs. La première est utilisée pour la couverture, mais il est possible d'en changer sur cette page en cliquant sur le vu (√).

Sixième étape : saisir un auteur

Sur l'onglet caractéristiques sur la dernière colonne vis à vis du libellé Auteur saisissez un nom.

Important : le champ utilisateur lié, sera utilisé pour les abonnements des instituts. Ne pas utilisez !
