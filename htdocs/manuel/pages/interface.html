Interface

En général

Lors de la connexion vous vous retrouvé dans l'espace réservé aux employés de la boutique, aussi appelé Backoffice.

L'image ci-dessous présente les trois principaux aspects de l'interface.

Description

  • Le centre de notifications vous signale quand il y a une nouvelle commande, un nouveau client ou un nouveau message d'un client
  • La zone de recherche vous permet de trouver n'importe quoi n'importe où sans devoir passer par la navigation
  • La barre principale vous permet d'accéder à toute la gestion de votre boutique

La barre principale

Le catalogue

Le catalogue va vous permettre de gérer tout ce qui est propre à vos produit. Vous pourrez créer, modifier ou supprimer vos articles. Vous pourrez gérer les catégories pour ranger vos articles.

C'est également ici que pourront être gérés les fichiers joints publics. Les fichiers joints privés (accessibles après achat) sont manipulables dans le détail de la fiche produit.

Commandes

Dans cette rubrique se trouveront toutes les commandes listées. C'est là que les commandes pourront être créées et modifiées. C'est également ici que l'état des abonnements sera géré

Clients

Dans cette rubrique se trouvent tous les clients ayant créé un compte, qu'ils aient ou non acheté quelque chose. Leur compte peut être modifier depuis cette rubrique. Le paniers abandonnés pourront aussi être consultés dans cette page.

Promotions

Gère les prix spéciaux, notamment les prix différents pour chaque devise ou les bons de réduction accordés au clients.

Localisation

Ce sont principalement les traductions qui seront accessibles depuis ce menu.

Modules

Nombre de fonctionnalités, comme p.ex la gestion de la page d'accueil, des news, des événements et du menu principal, passent par l'utilisation des modules.

Préférences

Gestion des pages texte. La gestion des menu relatives ces pages de texte se fait via les modules

Paramètres avancés

Afin d'assurer ses arrières, en cas de perte de données, il est possible de sauvegarder manuellement sur demande la base de données de la boutique.

Statistiques

Toutes sortes de statistiques relatives à vos visiteurs et à vos ventes.
